スキル

AI開発を中心に、Webアプリケーションの設計・実装も行っています。

ディープラーニングの基本原理

『ゼロから作るDeep Learning』を通して、ニューラルネットワークの順伝播・逆伝播、活性化関数、損失関数といった基本原理を理解しています。

Python

AI開発の基盤となるPythonの文法・標準ライブラリを理解し、データ処理やモデル実装に応用できます。

PyTorch

『PyTorchによる発展ディープラーニング』を通して、モデル構築、データセットの前処理、学習ループの実装などを実践。AIフレームワークとして使いこなしています。

時系列データ分析

Bitcoinの値動きという「時系列データ」を対象にAI予測モデルを開発。特徴量設計・モデルチューニング・検証の一連の流れを経験しています。

予測モデルの構築

データ収集、前処理、学習、推論までのプロセスを自力で構築。LightGBMやscikit-learnを使い分け、AI予測をWebアプリに組み込む実装経験があります。

Web開発

フロントエンド（HTML / CSS / JavaScript / React）およびバックエンド（Python / Node.js / PHP）の基礎を学習済み。FlaskやAPI連携を通じてAIモデルをWeb上で動作させる実装が可能です。
